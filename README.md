# CPSC-571-Group-Project

### Python Packages to Install
- Flask

### Run (Windows)
- Go to repository with the website_main.py
- Run:
    - ```set FLASK_APP=website_main.py```
    - ```flask run```
    
### Run (MAC)
- Go to repository with the website_main.py
- Run:
    - ```export FLASK_APP=website_main.py```
    - ```flask run```
    
### See webpage
- http://127.0.0.1:5000/
